RPCS3

An open-source Sony PlayStation 3 emulator written in C++.

Download » Sources »

{% endblock %} {% block content %}

Goals

  • Create an easy platform for developers to test their PS3 apps and homebrews.
  • Centralize a great part of the knowledge about the hardware and software of the PlayStation 3.
  • Play PS3 games and have fun!

Features

  • Cell processor interpreter.
  • Basic RSX emulation.
  • Basic SysCalls implemented for HLE.
  • Loading simple homebrews.
